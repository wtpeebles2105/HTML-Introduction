Basic's of Coding

Understanding code

Code is basically the concept of giving commands to a computer so that will execute those commands in the exact order you put them in.

What is HTML?

HTML stands for Hypertext Markup Lanuage. It is widely used on internet pages along with javascript.

How code works on the internet

The internet is made up of tons of HTML pages which are linked together with a protocol called HTTP. When browsing from your computer you use the internet to pull HTML pages from repositories such as servers.

Tags and Elements

Elements are what make up HTML documents. Tags are the way we tell the computer which elements we are using in our code.

Inline and Block Elements

There are two types of elements in an HTML file. Inline happen inside the current line you are coding in. Block elements form an "invisible box" around the contents which can cause undesired breaks in sentences.

Void Tags

Void Tags are tags that don't require a closing tag such as the "img" tag.

Break vs Pargraph

Paragraph is a block element. It puts an invisible box around text and wraps the text as needed. Break is an inline element. When using breaks it stops on its current line right after you close it.
